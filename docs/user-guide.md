## 系统结构

系统分为三个操作端，对应保理业务中的不同角色：

**银行管理端** — 系统基础配置（标签、产品、定价规则）

**供应商端** — 融资企业日常操作（注册、额度申请、融资、费用查询）

**资金方端** — 保理公司全流程管理（客群、额度审批、业务、结算、账户、市场经营）

---

## 功能清单

### 银行管理端

| 功能 | 说明 |
|------|------|
| 标签管理 | 客户维度 / 融资维度标签的增、删、改、启停。标签用于动态定价和风控分层 |
| 产品管理 | 22 个产品 SKU 的完整配置，含费项明细（计费方式、付费方式、是否开票）、担保配置、中登登记环节 |
| 产品定价 | 按标签维度设定上浮/下浮规则，按产品设定基准价格，两者联动生成最终定价 |

### 供应商端

| 功能 | 说明 |
|------|------|
| 驾驶舱 | 额度总额、额度余额、融资余额统计，最近融资记录，到期预警提醒 |
| 注册管理 | 三步注册向导：手机验证 → 企业/法人/操作人信息填写 → 附件上传（含 OCR 自动识别模拟）+ 协议签署 |
| 我的额度 | 额度列表（含使用率进度条），可查看8级审批进度、子额度、费用明细、交易对手信息 |
| 业务管理 | 融资申请列表，含全流程状态跟踪 |

### 资金方端

| 功能 | 说明 |
|------|------|
| 客户管理 | 供应商列表，支持搜索、状态筛选，详情含基本信息/标签/联系方式 |
| 核心企业额度 | 按核心企业维度管理授信额度，支持循环/非循环设置 |
| 供应商额度 | 8级审批流程可视化（供应商经办→复核→客户经理→市场总监→风险复核→风险总监→审查委员会→总经理），含子额度、费用、交易对手详情 |
| 业务管理 | 池保理、预保理、国内保理-补、票据保理-补四类业务的列表和详情 |
| 结算管理 | 票据兑付、尾款划转、退息划转三个模块集中管理 |
| 账户管理 | 账户余额一览，可按账户查看交易流水明细 |
| 市场经营 | 业务指标 KPI 统计、客户营销拜访记录管理 |

## 操作建议

以下为建议的验证路径，可完整体验核心业务闭环：

### 路径一：供应商注册

1. 切换到"供应商"端 → 注册管理
2. 填写三步注册表单 → 提交
3. 切换到"资金方"端 → 客户管理 → 查看新注册的供应商

### 路径二：额度 → 融资

1. 供应商端 → 我的额度 → 点击"查看"任一额度
2. 切换到"资金方"端 → 供应商额度 → 查看同一笔额度的8级审批进度
3. 回到"供应商"端 → 业务管理 → 查看融资列表

### 路径三：结算闭环

1. 资金方端 → 结算管理 → 票据管理 Tab → 点击"兑付"操作
2. 切换"尾款划转"Tab → 确认划转
3. 账户管理 → 查看对应的交易流水

## 数据说明

- 当前为演示数据，存储在浏览器本地（localStorage），刷新不会丢失
