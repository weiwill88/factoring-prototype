## 系统UI双轨特征

通过截图分析，系统存在**两套明显不同的UI风格**：

1. **新版UI**（img_007~img_031, img_065）：
   - 白底蓝色主题
   - 现代化卡片式布局
   - 圆角按钮、Tag标签
   - 三步注册向导等
   - **主要覆盖**：供应商端功能

2. **旧版UI**（img_036~img_064）：
   - 传统后台管理风格
   - 紫色/橙色主题按钮
   - 经典表格布局
   - **主要覆盖**：资金方端功能

这说明系统经历了前端迭代升级，供应商端已用新框架重写，资金方端仍在使用旧版。
